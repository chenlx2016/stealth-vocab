# Stealth Vocab 扩展加载说明

## 📋 加载步骤

### 1. 打开Chrome扩展页面
- 在Chrome地址栏输入：`chrome://extensions/`
- 或者点击菜单 → 更多工具 → 扩展程序

### 2. 启用开发者模式
- 在扩展页面右上角找到"开发者模式"切换开关
- 点击启用开发者模式

### 3. 加载扩展
- 点击"加载已解压的扩展程序"按钮
- 选择当前项目文件夹（包含manifest.json的目录）
- 点击"选择文件夹"

### 4. 验证安装
- ✅ 在扩展列表中应该看到"Stealth Vocab"
- ✅ 浏览器工具栏应该显示扩展图标（深色圆形，带"V"字母）
- ✅ 扩展详情显示版本 1.0.0

### 5. 测试Side Panel
- 点击扩展图标或右键选择"Open Stealth Vocab"
- 应该打开侧边栏，显示深色主题的词汇学习界面
- 界面应包含：
  - 顶部进度指示器："今日已复习 0 个词"
  - 中间词汇卡片：显示"加载中..."
  - 底部三个反馈按钮：✅掌握 / ❓模糊 / ❌遗忘

### 6. 测试快捷键
- 按下 `Ctrl+Shift+V`（Windows/Linux）或 `Command+Shift+V`（macOS）
- 在浏览器控制台应该看到"Toggle sidepanel command received"日志

## 🔍 故障排除

### 如果扩展加载失败：
1. 检查manifest.json语法是否正确
2. 确认所有文件路径存在
3. 查看扩展页面的错误信息

### 如果Side Panel无法打开：
1. 检查浏览器是否支持Side Panel API（需要Chrome 114+）
2. 查看浏览器控制台是否有JavaScript错误

### 如果图标不显示：
1. 确认icons/目录中的三个PNG文件存在
2. 检查文件权限是否正确

## ✅ 预期结果

成功加载后，你应该能够：
- 在chrome://extensions/页面看到扩展
- 通过图标或快捷键打开Side Panel
- 看到深色低对比度的学习界面
- 点击卡片可以翻面（虽然功能尚未完全实现）
